# Thư mục hình ảnh kiến trúc

Thư mục này chứa các hình ảnh sơ đồ kiến trúc được tạo từ draw.io.

## Các file hình ảnh cần có:

1. **vpc-architecture-initial.png** - Kiến trúc ban đầu (không có NAT Gateway)
2. **vpc-architecture-no-private-internet.png** - Kiến trúc không có internet access cho private subnet
3. **vpc-architecture-ssh-access.png** - SSH Access từ App đến DB
4. **vpc-architecture-nat-gateway.png** - Kiến trúc với NAT Gateway (hiện tại)

## Hướng dẫn export từ draw.io:

1. Mở file draw.io của bạn
2. Chọn diagram cần export
3. File → Export as → PNG (hoặc SVG)
4. Chọn độ phân giải phù hợp (khuyến nghị: 2x hoặc 3x cho chất lượng tốt)
5. Lưu file vào thư mục này với tên tương ứng

## Format khuyến nghị:

- **Format**: PNG (cho README) hoặc SVG (cho chất lượng tốt hơn)
- **Kích thước**: Tối thiểu 1200px width cho PNG
- **Background**: Nền trắng hoặc trong suốt
